## 🚀 快速部署指南

### 方式1：Vercel（推荐）
1. 访问 [vercel.com](https://vercel.com)
2. 用GitHub账号登录
3. 上传这个项目文件夹
4. 点击部署，几分钟后就能获得永久链接

### 方式2：GitHub Pages
1. 在GitHub创建新仓库
2. 上传所有文件（index.html, styles.css等）
3. 在仓库设置中启用GitHub Pages
4. 获得类似 `yourname.github.io/zhengzhikang` 的链接
